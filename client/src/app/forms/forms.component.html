Instruções

Este ambiente foi desenvolvido para possibilitar os usuários de realizar o download dos limites e outros dados de propriedades dos imóveis cadastrados no CAR (Cadastro Ambiental Rural), por meio de um acesso interno aos dados públicos do portal do SICAR (Sistema Nacional de Cadastro Ambiental Rural).

  • * Preencha todos os campos corretamente.
  • * A tabela inserida deverá estar no formato csv.
  • * Para realizar o download é necessário inserir uma tabela em formato csv, com pelo menos uma coluna que contenha os códigos de propriedades (COD_IMOVEL) desejados.
  • * Ao final do processo um link de download será enviado para o e-mail informado.

Importante: O sistema irá considerar que a primeira coluna tenha o código da propriedade. Ou seja, em sua tabela configure a coluna que contém o código do imóvel na primeira coluna(A) da tabela.
